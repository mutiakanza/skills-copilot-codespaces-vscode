# Sistem Pembelajaran dan Penilaian Perguruan Tinggi

Web aplikasi untuk manajemen pembelajaran dan penilaian di perguruan tinggi yang membantu dosen dan mahasiswa mengelola mata kuliah, tugas, dan nilai.

## ğŸ“š Fitur Utama

### 1. Dashboard
- **Statistik Real-time**: Menampilkan total mata kuliah, tugas aktif, rata-rata nilai, dan progress pembelajaran
- **Aktivitas Terkini**: Log aktivitas terbaru dalam sistem
- **Ringkasan Visual**: Kartu statistik yang mudah dibaca

### 2. Manajemen Mata Kuliah
- Tambah mata kuliah baru dengan informasi lengkap:
  - Nama mata kuliah
  - Kode mata kuliah
  - Nama dosen pengampu
  - Jumlah SKS
- Hapus mata kuliah
- Tampilan kartu yang informatif

### 3. Sistem Penilaian
- Berbagai jenis penilaian:
  - Tugas
  - Quiz
  - UTS (Ujian Tengah Semester)
  - UAS (Ujian Akhir Semester)
  - Praktikum
- Status penilaian otomatis (Selesai, Hari Ini, Akan Datang)
- Bobot penilaian dapat disesuaikan
- Tenggat waktu pengumpulan

### 4. Manajemen Nilai
- Tampilan tabel nilai mahasiswa
- Pengelompokan berdasarkan mahasiswa dan mata kuliah
- Perhitungan rata-rata nilai otomatis

## ğŸš€ Cara Menggunakan

### Instalasi
1. Clone repositori ini
2. Buka file `index.html` di browser

### Penggunaan Dasar

#### Menambah Mata Kuliah
1. Klik menu "Mata Kuliah" di navigasi
2. Klik tombol "+ Tambah Mata Kuliah"
3. Isi formulir dengan informasi mata kuliah
4. Klik "Simpan"

#### Menambah Penilaian
1. Klik menu "Penilaian" di navigasi
2. Klik tombol "+ Tambah Penilaian"
3. Pilih mata kuliah
4. Isi judul, jenis, tenggat waktu, dan bobot penilaian
5. Klik "Simpan"

#### Melihat Nilai
1. Klik menu "Nilai" di navigasi
2. Lihat tabel nilai mahasiswa yang terorganisir
